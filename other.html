Czym jest GIT?

Jest to system kontroli wersji. Służy do śledzenia i wersjonowania zmian.

Podstawowe komendy GIT

git fetch

służy do sprawdzenia różnicy między repozytorium zdalnym a lokalnym

git push

służy do wrzucenia zmian na zdalne repozytorium

git commit -m "tytuł zmian"

służy do dodania zmian lokalnych do repozytorium
